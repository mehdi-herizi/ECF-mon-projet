# Nexary

# Catalogue Produits

Site catalogue développé en JavaScript permettant d’afficher
des produits depuis un fichier JSON avec un système de catégories.

## Fonctionnalités
- Affichage dynamique des produits
- Tri par catégories
- Données chargées depuis un fichier JSON
- Interface responsive

## Technologies
- HTML
- CSS
- JavaScript
- JSON

## Utilisation
Ouvrir `index.html` dans un navigateur.
